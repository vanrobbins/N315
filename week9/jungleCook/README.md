### Installation

1. Clone the repository:

```bash
git clone https://github.com/vanrobbins/N315.git
cd jungleCook
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run serve
```

4. In a separate terminal, watch SCSS files for compilation:

```bash
npm run compile:sass
```
